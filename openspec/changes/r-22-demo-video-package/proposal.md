# r-22-demo-video-package

## Summary
Add a repository-contained demo-video preparation package for the course submission. The package provides an animated slide deck, deterministic product screenshots, an English narration script, optional OpenAI TTS voiceover generation, and recording guidance for a 1-2 minute demo.

## Why
The course submission requires a short video that shows the product and explains how it was built agentically. Keeping the demo assets in the repository makes the submission repeatable, reviewable, and aligned with the project's evidence-first development process.

## Scope
- Add a static slide deck under `docs/demo-video/`.
- Capture product screenshots from local fixture data, not production data.
- Provide an English voiceover script and generated MP3 voiceover.
- Document recording steps and AI voice disclosure.
- Record process evidence in `docs/development_process.md` and `docs/state.md`.

## Out of Scope
- Publishing the final video to an external platform.
- Creating external accounts or spending beyond already available API access.
- Claiming production readiness from the demo package.
