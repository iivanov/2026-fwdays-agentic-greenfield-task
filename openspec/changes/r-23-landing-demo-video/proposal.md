# r-23-landing-demo-video

## Summary
Embed the generated project demo video on the unauthenticated public landing page so visitors can watch the product and agentic-development walkthrough directly from the browser app.

## Why
The course submission asks for a short demo video. The repository already contains the rendered MP4; placing it on the landing page makes the demo discoverable from the deployed product instead of requiring users to browse repository files.

## Scope
- Serve the rendered MP4 as a Vite public browser asset.
- Add a responsive demo-video section to the public landing page.
- Add Playwright smoke coverage for the landing section and served MP4.
- Sync public landing and demo-video package specs.
